🔁 1. Handle Complaints

Login → Admin Dashboard → View Unresolved Complaints
    ↓
Click Complaint → See Details
    ↓
[Assign to Solver] → Select Solver → Confirm
    ↓
Track Status (Live)

🧑‍🔧 2. Manage Solvers

Login → Dashboard → Solvers Tab
    ↓
View Solver List / Add New Solver
    ↓
Assign Complaints → View Assigned Tasks
    ↓
Reassign / Remove Solver (if needed)

👥 3. Monitor Users (Citizens)

Login → Dashboard → Citizens Tab
    ↓
Search / Filter by Area
    ↓
View Profile / Activity Log
    ↓
Issue Warning or Reward

🛡️ 4. Manage Volunteer Army

Dashboard → Volunteer Army Section
    ↓
View All Armies → Click One
    ↓
Approve / Reject New Join Requests
    ↓
Send Mission Updates

🎖️ 5. Distribute Rewards

Dashboard → Rewards Panel
    ↓
Select Criteria (e.g., Top Solver, Most Active Citizen)
    ↓
Assign Points / Badges → Confirm

🗣️ 6. Moderate Community Posts

Dashboard → Forum Moderation
    ↓
View Flagged Posts / Comments
    ↓
Review Content → Approve / Delete / Warn

📢 7. Send Announcements

Dashboard → Broadcast Section
    ↓
Create New Announcement
    ↓
Select Audience (All / Zone-wise / Role-based)
    ↓
Send → Appear in Notifications

🔔 8. Manage Notifications

Dashboard → Notifications
    ↓
View Sent Notifications
    ↓
Create / Schedule New One

📊 9. View Analytics Dashboard

Dashboard → Analytics
    ↓
View Complaint Trends, Solver Efficiency, Army Activity, etc.

👤 10. Update Admin Profile

Dashboard → Profile
    ↓
Edit Details → Change Password → Save
