📝 1. Report a Complaint

Login/Register
    ↓
Dashboard → Click "Report Issue"
    ↓
Fill Form (Title, Description, Category, Photo, Location)
    ↓
Submit → Redirect to Complaint Status Page

📋 2. View Complaint History

Dashboard → My Complaints
    ↓
List View → Filter by Status (Open, In Progress, Resolved)
    ↓
Click Complaint → View Full Details

⭐ 3. Give Feedback on Resolved Complaint

Complaint Detail Page → If Resolved
    ↓
Click "Give Feedback" → Submit Rating + Comment

💬 4. Use Community Forum

Dashboard → Click "Community"
    ↓
View Posts
    ↓
[Option A] Create Post → Submit  
[Option B] Comment / Like / Report on Posts

🛡️ 5. Join Volunteer Army

Dashboard → Click "Volunteer Army"
    ↓
Browse Available Armies → Click One
    ↓
Fill Join Form → Submit → See Join Status

👤 6. Manage Profile

Dashboard → Click "Profile"
    ↓
View Profile → Click "Edit"
    ↓
Update Info (Bio, Photo, Contact) → Save → View Updated Profile

🔔 7. View Notifications

Dashboard → Click "Notifications"
    ↓
View Notification List → Click to Open

🆘 8. Access Help / Support (Optional, but recommended)

Dashboard → Click "Help"
    ↓
See FAQs / Contact Form → Submit Query

⚙️ 9. Change Settings (Optional for future scaling)

Dashboard → Settings
    ↓
Update Language, Alert Preferences → Save

✅ Summary Table of Citizen Flows
Flow	Description
Report Complaint	With form and status tracking
View Complaint History	Filterable list of past complaints
Give Feedback	Rate & review resolved complaints
Community Forum	Posts, comments, likes
Volunteer Army	Join and track army groups
Profile Management	View/edit profile
Notifications	View updates
Help/Support	Ask questions or get help
Settings (Optional)	App preferences
